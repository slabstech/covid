  * Intall addon


* Setting up pgp key for signed email

  * sudo apt-get install gnupg

  * gpg --gen-key

  * gpg --armor --export uni-mail-id > public-key.asc

  * open pgp.mit.edu

  * copy the full text from public-key.asc and paste it in textareabox
